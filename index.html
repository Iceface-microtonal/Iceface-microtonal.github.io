Iceface Tuningとは

163 +

164 + Iceface Tuning は、日本の作曲家 H. Wakabayashi によって提案された独自の微分音チュ + ーニングです。 165 + ハ長調を基準とした場合、すべての黒鍵（シャープ/フラット）を約 50セント上げた音（四分音）として「アイス・ノート（Ice Note）」に割り当てます。 166 +

167 +

168 + これにより、旋律や和音に対して局所的なデチューン効果（ コーラス感）、またはダブリング的な効果が生まれ、 169 + 通常の12平均律の中に自然に微分音を融合させることが可能になります。 170 +

171 +

172 + 派生として、「Iced Lydian」「Iced Maqam Rast」などの"アイス化されたモード + "もあり、 173 + 非西洋音楽にも応用が広がっています。 174 +
